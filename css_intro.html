CSS: Cascading Style Sheet.

Ways to add CSS to HTML

  1. Inline: Any CSS code written within the HTML tag itself. This is having HIGH Priority.
  2. Internal: Any CSS code written inside the HEAD tag using STYLE tag is internal CSS.
  3. External: Any CSS written in a Seperate file & imported within the current HTML file is called External CSS.
